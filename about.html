About this project

I came up with the idea of developing this website when I visited a froyo shop in Hyderabad. i realized that while the shop and the froyo was great, the shopped lacked a website for enhancing their marketing. So I thought, 'let me try and make a website for a froyo shop!'. And here it is.

I took some time and good amount of studying user experience. The colour theme has been chosen keeping in mind desserts and sweets. The website has been designed in such a way that any user finds it easy to navigate through the pages. The logo and icons were designed by me using Adobe Photoshop.

About me

I am a UI/UX Developer at Deloitte US-India. Passionate about UX and Human-computer Interaction.

Technologies used for this website

I used HTML, CSS3, JavaScript, jQuery, Adobe Photoshop for developing this website.
